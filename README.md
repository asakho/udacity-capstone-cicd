# CAPSTONE
## _Udacity Cloud Devops Engineer Nanodegree_

## Project Overview

- Create EKS infrastructure using cloudformation
- Deploy blue/green simple web page
- based on Jenkins CI


## Installation

Launch EC2 instance 

- Install the dependencies.

```sh
python3-pip
aws cli
docker
jenkins
kubectl
eksctl
lint
```
- Install jenkins plugins.
```sh
kubernetes
blue ocean
pipeline aws steps
```
